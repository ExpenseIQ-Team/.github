# 💰 ExpenseIQ – Money Management System

A full-stack web application designed to help users efficiently manage their personal finances by tracking income, expenses, savings, and overall financial activity in one place.

## 🚀 Tech Stack

- Frontend: React.js  
- Backend: Java, Spring Boot  
- Database: MySQL  
- API: RESTful Web Services  

---

## 📌 Project Overview

ExpenseIQ is built to simplify personal finance management through a user-friendly interface and efficient backend processing.

Users can:
- Add and manage income and expenses  
- Categorize transactions (Food, Travel, Bills, etc.)  
- Monitor savings and financial activity  
- View summarized insights of their spending  

---

## ⚙️ How It Works

### 🔹 Frontend
- Built using React.js  
- Provides an interactive UI for users  
- Allows adding and categorizing transactions  
- Displays summaries and reports  

### 🔹 Backend
- Developed using Spring Boot  
- Handles business logic and calculations  
- Provides REST APIs for communication  

### 🔹 Database
- Stores all financial data securely  
- Supports efficient data retrieval and updates  

---

## 🔧 Key Features

- Add, update, and delete transactions (CRUD operations)  
- Category-wise expense tracking  
- Monthly spending analysis  
- Real-time balance calculation  
- REST API integration  
- Clean and responsive UI
