# 💰 ExpenseIQ – Money Management System

**ExpenseIQ** is a full-stack web application designed to help users efficiently manage their personal finances by tracking income, expenses, savings, and overall financial activity in one place.

---

## 🚀 Tech Stack

- React.js  
- Java (Spring Boot)  
- MySQL  
- REST APIs  

---

## 📌 About Project

ExpenseIQ provides a simple and user-friendly platform where users can add, manage, and categorize their financial transactions such as food, travel, and bills.

The frontend is built using React.js, allowing users to interact with the system seamlessly, while the backend developed with Spring Boot handles business logic, processes data, and communicates with the database through REST APIs.

---

## 🎯 Objective

To build an efficient and scalable financial management system that demonstrates full-stack development concepts including CRUD operations, API integration, and database management.
